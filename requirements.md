# Requirements
- I want to build a markdown WYSIWYG editor extension for VSCODE

## Features
- User can right click files and click on "Open with Markdown Plus"
- This opens up a `What you see is what you get` pane where you can see and edit markdown files in realtime and it shows up in markdown style + file also gets updated

- Think of this as a Vscode Markdown editor plus the preview mode
- Highly inspired by "Obsidian" which has a similar markdown viewer

## Some important features
- This extension should be light weight and fast when opening and editing markdown files
- There should be as little lag as possible in what I see ( in the UI) and update the file
